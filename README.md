# Mamoor Ahmad — Portfolio

AI & Full-Stack Developer Portfolio. 28+ projects shipped.

🔗 **Live:** https://mamoor123.github.io/portfolio/

## Tech Stack

- Pure HTML/CSS/JS (no build tools, no framework)
- GSAP 3.12.5 + ScrollTrigger for animations
- Inter + JetBrains Mono fonts
- GitHub Pages hosting

## Features

- 🌙 Dark/Light theme with localStorage persistence
- 📱 Fully responsive (mobile, tablet, desktop)
- ♿ Accessible (skip links, focus styles, aria labels, reduced motion)
- 🤖 Embedded AI chatbot (regex-based intent matching)
- 📊 Live GitHub contribution graph
- 🔍 SEO optimized (Schema.org, OG tags, sitemap)
- ✨ Scroll animations, typing effect, custom cursor

## Structure

